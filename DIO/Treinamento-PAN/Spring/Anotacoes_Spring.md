
# Spring Framework
    Framework open source desenvolvido para o JAVA nos padrões de projeto inversão de controle e injeção de dependências.

## Inversão de Controle
    Agora entrará o Container onde ficará responsável em realizar o gerenciamento de Objetos do programa. Assim quando iniciar a aplicação, usaremos somente o objeto que já estará criado pelo Containner. Além disso a finalização desse será de responsabilidade dele. Delegando as funções necessárias para o container.

## Injeções de independências
    Injeção de Dependência é um padrão de projeto que ajuda muito a deixar o código desacoplado, melhora a legibilidade e interpretação do código, melhora a distribuição de responsabilidades entre as classes e facilita a manutenção do código.

## Beans
    Objeto que é instanciado e controlado pelo Container.
